Promise

class Promise<T>

Simple promise utility class Usage:

     let promise = Promise<UIImage>(on: .background)
     someSyncTask() { result, error in
         if let error = error {
             promise.reject(error)
         } else {
             promise.resolve(result)
         }
     }
